# FrenchMB
Enriched AZERTY Windows keyboard layout with liturgical latin characters.

Created with MS Keyboard Creator 1.4 for Windows 10/11.

French Guillemets = AltGr+Q and AltGr+M

English Guillemets = AltGr+Shift+Q and AltGr+Shift+M

Apostrophe = Shift+Â²

Unaccentend OE and AE ligatures = AltGr+a, AltGr+Shift+A, AltGr+o, AltGr+Shift+o

Accented OE (lowercase only) and AE ligatures = AltGr+z, AltGr+Shift+Z, AltGr+p
